### Выводы

JavaScript гибче C# — можно менять типы переменных во время выполнения

JavaScript проще синтаксически — меньше "церемоний" при написании кода

JavaScript опаснее — ошибки обнаруживаются только во время выполнения, компилятор не помогает

C# безопаснее — строгая типизация и компилятор предотвращают множество ошибок на этапе разработки

Области применения разные — C# для backend, десктопных приложений и игр; JavaScript для frontend и веб-разработки

Node.js расширяет возможности JS — позволяет использовать JavaScript для серверной разработки

Важно использовать строгое сравнение (===) — чтобы избежать неожиданного поведения из-за автоматического преобразования типов
